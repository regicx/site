Resumo do cronograma para cabelos ondulados:

  • Hidratação (H): Segunda e Sexta
  • Nutrição (N): Quarta
  • Reconstrução (R): Sexta (a cada 2 semanas)

    • Dicas: Use produtos adequados, evite lavar o cabelo diariamente e finalize com creme ou leave-in para controlar frizz e definir as ondas.


    Para cabelo crespo:


  • Hidratação (H): Segunda e Sexta
  • Nutrição (N): Quarta
  • Reconstrução (R): A cada 2 semanas (Quarta)

    • Dicas: Faça umectação com óleos vegetais, lave com menos frequência e use produtos sem sulfato. Finalize com cremes ou óleos para frizz e definição.


    Cronograma para cabelo cacheado:

  • Hidratação (H): Segunda e Sexta
  • Nutrição (N): Quarta
  • Reconstrução (R): A cada 2 semanas (Sexta)

    • Dicas: Use shampoo sem sulfato, faça umectação com óleos nos dias de nutrição e finalize com creme ou ativador de cachos.


    Cronograma para cabelo liso:

  • Hidratação (H): Segunda e Sexta
  • Nutrição (N): Quarta
  • Reconstrução (R): A cada 2 semanas (Sexta)

    • Dicas: Use shampoo para controle de oleosidade, óleos leves para nutrição e protetores térmicos para secador ou chapinha. Prefira produtos leves para manter o volume.
